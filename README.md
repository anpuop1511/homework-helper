# Homework Helper

A homework management app with an AI study buddy, social features, and gamification.

## Features

### Homework & Productivity
- Track assignments and due dates
- Organize school work in one place
- Designed for quick daily check-ins

### AI Study Buddy
- Built-in AI chat to help you study and explain topics
- Optional **Bring Your Own API key** support for AI-powered features (configured in Settings)
- Google Classroom “paste import” (AI-powered) to extract class names + subjects from copied Classroom text

### Classes & Subjects
- Create/manage classes
- Link classes to subjects
- Customize subject names in Settings (example: rename **Science** → **Bio**)

### Social
- Friends system with requests + pending states
- Friend interactions designed to update smoothly with Firestore

### Gamification
- Battle Pass style progression UI
- Reward/progress experience designed to keep you consistent

### Navigation & UI
- Polished Settings design improvements
- Customizable bottom navigation bar (with **Home always pinned**)
